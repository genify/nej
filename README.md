NEJ - JavaScript Framework
====================================

跨平台WEB前端开发框架，主要特性包括：

* [依赖管理系统系统](./doc/DEPENDENCY.md)支持
* 多平台适配（浏览器、移动APP、桌面APP等）
* 多方案集成（模板化、可组合的模块化开发、单页系统按需载入、基于配置的跨域异步请求等）
* 可自由定制的产品发布（按平台定制、按功能定制）
* 丰富的可灵活扩展组件系统（可验证表单、列表、拖拽、滑块、日历、富文本编辑器等）
* 工具支持（NEJ工具集、NEJ脚手架等）
* 新技术整合（对于高端目标平台自动应用新技术）
