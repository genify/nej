nej.$()._$filter(<Function|String>)

文件标识：{lib}util/chain/NodeList.js
依赖列表：{lib}util/query/query.js,{lib}base/element.js,{lib}base/util.js
平台支持：          

• 简介

类似于ES5的Array#filter, 一个过滤, 即过滤_$$NodeList的所有节点集并筛选符合的节点比如:
1
2
3
4
5
6
7
// 返回节点集中的匹配选择器.strong:nth-child(3n)的节点
$("li")._$filter(".strong:nth-child(3n)")
 
// 相当于  ===>
$("li")._$filter(function(_node){
    return $(_node)._$matches(".strong:nth-child(3n)");
});
注意callback中传入的节点是裸节点，而不是包装后的_$$NodeList

• 输入

类型：Function|String
描述：_fn 遍历函数，接受两个参数，当前遍历到的节点和节点下标。同时也接受一个Selector，筛选出节点集中满足选择器的节点

类型：_$$NodeList
