nej.e._$bindCopyAction(<String|Node>,<String|Function>)

• 简介

绑定复制操作，服务器放置剪切板操作Flash nej_clipboard.swf如果flash文件不在/res/下可以通过以下方式配置
1
2
3
4
// 在引入define.js之前配置NEJ
window.NEJ_CONF ={
    clipboard:'/other/path/nej_clipboard.swf'
};
使用举例页面结构：
1
2
<!-- 复制按钮 -->
<input id="copyBtn" type="button" value="复制"/>
脚本绑定复制功能：
1
2
3
4
5
6
7
// 或者通过参数在绑定时输入要复制的内容
nej.e._$bindCopyAction('copyBtn','text to clipboard');
// 如果要复制的内容动态生成
nej.e._$bindCopyAction('copyBtn',function(){
    // 此函数必须同步返回结果
    return 'text to clipboard'
});

类型：String|Node
描述：操作节点
类型：String|Function
