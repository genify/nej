nej.$()._$prev(<String>,<Boolean>)

文件标识：{lib}util/chain/NodeList.js
依赖列表：{lib}util/query/query.js,{lib}base/element.js,{lib}base/util.js
平台支持：          

• 简介

与_$parent类似,查找 所有节点 的第一个(或所有根据_all参数)满足关系的 前序兄弟节点 (previousSibling)集, 并返回$NodeList
1
2
3
4
$("td")._$prev("th[scope=row]", true)
// 返回所有在td之前的th元素, 它们的scope属性为 row        
$("td")._$prev("th[scope=row]")
// 只返回直接相邻的前节点，如果不满足选择器则返回空节点集

• 输入

类型：String
描述：_selector 选择器
类型：Boolean
描述：_all 是否获取所有前序节点

• 输出

类型：[type]
