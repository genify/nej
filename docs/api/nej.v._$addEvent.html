• 输入

类型：String|Node
描述：节点ID或者对象
类型：String
描述：事件类型，不带on前缀，不区分大小写
类型：Function
描述：事件处理函数
类型：Boolean
描述：是否捕获阶段事件，IE低版本浏览器忽略此参数
