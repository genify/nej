nej.e._$html2node(<String>)

文件标识：{lib}base/element.js
依赖列表：{lib}base/constant.js,{lib}base/event.js,{lib}base/util.js,{patch}api.js,{patch}json.js
平台支持：          

• 简介

html代码转节点对象，如果转换出来的节点数量超过[包含]2个，则最外面增加一个div节点，即返回的始终是一个节点
页面结构举例
1
2
3
<div id="abc">
  <span>123</span>
</div>
执行以下代码
1
2
3
var _e = NEJ.P("nej.e"),
    _html = '<div>1</div><div><span>2</span></div>';
var _node = _e._$html2node(_html)
返回以下结构
1
2
3
4
<div>
  <div>1</div>
  <div><span>2</span></div>
</div>

• 输入

类型：String
描述：代码

• 输出

类型：Node
描述：节点
