nej.$()._$html(<content>)

文件标识：{lib}util/chain/NodeList.js
依赖列表：{lib}util/query/query.js,{lib}base/element.js,{lib}base/util.js
平台支持：          

• 简介

获得节点集中的 第一个元素的value 或者设置所有元素的value(与_$text接口类似)Example:
1
2
3
4
$("input,textarea")._$val()
// 获取第一个满足'input,textarea'选择器元素的value值
$("title,h2")._$html("haha")
// 获得title(第一个元素)的innerHTML

• 输入

类型：content
描述：_content 要插入的内容

• 输出

类型：_$$NodeList|String
描述：setter操作返回_$$NodeList getter操作返回String
