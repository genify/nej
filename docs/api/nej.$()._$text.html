nej.$()._$text(<content>)

文件标识：E:/code/nej/src/util/chain/NodeList.js
依赖列表：
平台支持：          

• 简介

获得节点集中的 第一个元素的innerText 或者 设置所有元素的innerTextExample:
1
2
3
4
$("title,h2")._$text("haha")
// 同时设置title与h2的text内容为haha
$("title,h2")._$text()
// 获得title(第一个元素)的innerText

• 输入

类型：content
描述：_content 要插入的内容 , 不传入则认为是getter操作

• 输出

类型：_$$NodeList|String
描述：setter操作返回_$$NodeList getter操作返回String
